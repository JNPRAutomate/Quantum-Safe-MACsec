# Quantum Safe MACsec v3.2.2

Status: In Development

## Completed Changes

### Remove Profile Dependency

Status: Completed

Description

Removed the legacy `Profile` dependency used only for runtime performance measurements.

Removed:

- `import Profile`
- `prof = Profile.Profile(...)`
- `prof.start(...)`
- `prof.stop(...)`
- `prof.close(...)`

Reason

The profiler was not required for:

- ETSI QKD key retrieval
- KME communication
- MACsec CAK/CKN updates
- Junos configuration commits

Impact

No functional changes.

Validation

- Script imports successfully.
- Runtime logic unchanged.
- On-box and off-box execution paths preserved.


### Removed

- Removed unused imports:
  - Lock
  - hashlib
  - subprocess
  - ipaddress

### Impact

- No functional changes.
- Code cleanup only.

### Master KME Error Handling

Status: Completed

Description

Added explicit error handling when KME key retrieval fails.

Previous behavior:

- KME request could return None.
- Script continued execution.
- An undefined key variable could be referenced later.

New behavior:

- Script logs the error.
- Processing stops cleanly.

Impact

Improves robustness during KME communication failures.

Risk

Low.
